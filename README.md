# FR3D - ICML 2026

This is the repository that contains source code for the [FR3D website](https://fr3d-wm.github.io). The source code is heavily based on the [CUT3R website](https://github.com/CUT3R/CUT3R.github.io/tree/master). We thank the CUT3R authors for making their project page source code publicly available.
